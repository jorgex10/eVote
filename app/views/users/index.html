Bienvenido(a) a eVote

Aplicación web para gestionar el proceso de sufragio de una facultad perteneciente a la UNMSM

eVote trabaja con 3 tipos de usuarios: Admin, Miembros de mesa and Votantes:

eVote le permitirá realizar entre otros los siguientes procesos:

  • Cargar y gestionar votantes
  • Generar mesas de sufragio acordes con la cantidad de votantes
  • Generar miembros de mesa acorde a la cantidad de mesas de sufragio
  • Crear cuentas para los miembros de mesa
  • Buscar, generar y gestionar información de manera eficaz
  • Generar reportes
